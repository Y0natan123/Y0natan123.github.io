Word Memory App

A language learning application for memorizing words and tracking progress.

Features:

  • Multi-language pair support
  • Individual user progress tracking
  • Flashcard practice mode
  • Statistics and progress analytics
  • Mobile-responsive design
  • Admin interface for word management

This is a React-based application with a Node.js backend.

For development, run the application locally using:

npm run server (for backend)
npm run client (for frontend)
